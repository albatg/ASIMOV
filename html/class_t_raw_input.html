◆ loadFromFile()

bool TRawInput::loadFromFile ( )

Intenta cargar contenido desde un archivo a m_source.

Este método verifica que m_source esté configurada y no esté vacía. Luego intenta abrir el archivo indicado por m_source en modo de solo lectura. Si la apertura del archivo es exitosa, lee su contenido y lo almacena en m_source. Finalmente, marca m_loaded como verdadero y muestra un mensaje de éxito en la consola.

Returns
Devuelve true si la carga desde el archivo fue exitosa, false en caso contrario.
